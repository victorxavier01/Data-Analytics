# COVID-19 Data Exploration using SQL

## Objective
Analyze COVID-19 cases, deaths, and vaccination progress using SQL.

## Tools
- SQL Server (T-SQL)
- Public COVID-19 dataset

## Analysis Performed
- Total cases vs deaths (Brazil)
- Infection rate per population
- Countries with highest infection and death rates
- Global statistics
- Vaccination progress using:
  - CTE
  - Temporary Tables
  - Views

## Key Skills Demonstrated
- Aggregations
- Window Functions
- CTEs
- Temp Tables
- Views
- Data cleaning & filtering
